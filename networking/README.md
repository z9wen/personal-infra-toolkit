# Networking

Advanced network and edge-service automation used on personal VPS hosts.

| Path | Purpose |
| --- | --- |
| `xray/` | Deploy and operate an Xray-based TLS/QUIC edge service. |

The Xray project keeps its maintainable source modules under `xray/src/` and
publishes `xray-install.sh` in this directory for direct use on a server.
